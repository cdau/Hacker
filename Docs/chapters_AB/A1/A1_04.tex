\subsection*{Aufgabe 4}
Gegeben sei ein Merkmal X in einer Gesamtheit vom Umfang n mit arithmetischen Mittel x
und Standardabweichung s. Das Merkmal Y sei definiert durch: $y_{i}=\dfrac{x_{i}*\overline{x}}{s}$ für i = 1, ..., n
Zeigen Sie: das arithmetische Mittel von Y beträgt 0, und die Varianz von Y beträgt 1.
