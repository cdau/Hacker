\subsection*{Aufgabe 6}

(Klausur SS 2013)
Die Anzahl der Zugriffe pro Minute auf eine Datenbank sei (zumindest näherungsweise)
Poisson-verteilt mit einem bestimmten Parameter $\lambda$ > 0. Die Wahrscheinlichkeit, dass
innerhalb einer Minute genau ein Zugriff erfolgt, ist halb so groß wie die Wahrscheinlichkeit,
dass innerhalb einer Minute genau zwei Zugriffe erfolgen. Wie viele Zugriffe erfolgen im
Durchschnitt pro Minute?
