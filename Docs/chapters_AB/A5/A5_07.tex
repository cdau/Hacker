\subsection*{Aufgabe 7}

(Klausur WS 2013/14)
Die Anzahl der Zugriffe pro Minute auf eine Datenbank sei (zumindest näherungsweise)
Poisson-verteilt mit einem bestimmten Parameter $\lambda$ > 0. Die Wahrscheinlichkeit, dass
innerhalb einer Minute genau zwei Zugriffe erfolgen, ist größer als die Wahrscheinlichkeit,
dass innerhalb einer Minute genau drei Zugriffe erfolgen. Welche Werte kann $\lambda$ unter diesen
Umständen annehmen?
