\subsection*{Aufgabe 5}

(Klausur WS 2013/14)\newline
Eine stetige Zufallsvariable X besitze folgende Verteilungsfunktion $f:\mathbb{R}\to \mathbb{R}$: \newline
 
  F(x)=$\begin{cases}
 		0$ \hspace{50pt}für  x < 0 $\\
 		0,0625 \cdot x^4 $ \hspace{4pt}für $0\le x\le 2$ $\\
 		1$ \hspace{50pt}für x > 2 $\\
 		
 		\end{cases}$
 
\begin{enumerate}[leftmargin=0.7cm, label=\alph*)]
\item Mit welcher Wahrscheinlichkeit nimmt X einen Wert an, der größer als 1,5 ist?
\vspace{6cm}
\item Wie lautet der Median von X?
\vspace{5cm}
\item Wie lautet der Erwartungswert von X?
\end{enumerate}
