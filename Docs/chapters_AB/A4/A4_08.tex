\subsection*{Aufgabe 8}
Genau einer von sechs sehr ähnlichen Schlüsseln passt in eine bestimmte Tür. Sie probieren in zufälliger Reihenfolge die Schlüssel, bis einer passt. Wie viele Schlüssel müssen Sie im Durchschnitt ausprobieren? Wie ändert sich die Antwort, wenn Sie jedes Mal wieder aus allen sechs Schlüsseln zufällig auswählen (also \glqq mit Zurücklegen\grqq), zum Beispiel weil Sie den Schlüsselbund nach jedem Versuch aufgrund übermäßigen Alkoholkonsums fallen lassen?
