\subsection*{Aufgabe 14}

(Klausur SS 2013) 
Bei einem bestimmten Kartenspiel werden 32 Karten (darunter vier Asse) zufällig auf vier 
Spieler verteilt, d.h. jeder Spieler erhält 8 Karten. Geben Sie Verteilung und Erwartungswert der folgenden Zufallsvariablen an: 

\begin{enumerate} [label=\alph*)]
\item Zahl der Asse, die ein bestimmter Spieler bei einem Spiel erhält. 
\vspace{4cm}
\item Zahl der Spiele, die ein bestimmter Spieler spielen muss, bis er einmal alle vier Asse gleichzeitig erhält. 
\vspace{4cm}
\item Zahl der Spiele, die dieselben vier Spieler spielen müssen, bis irgendeiner der Spieler alle vier Asse gleichzeitig erhält.
\vspace{4cm}
\item Anzahl der Spiele, bei denen ein bestimmter Spieler kein einziges Ass erhält, wenn genau 50 Spiele gespielt werden.
\end{enumerate}
