\subsection*{Aufgabe 6}



Die natürlichen Zahlen von 1 bis n werden zufälligsortiert; der entstehende Vektor sei mit a = ($a_{1},a_{2},..., a_{n}$) bezeichnet. Ein Paar (i, j) $(1 \le i \le j \le n)$ mit ai > aj heißt eine Inversion. X sei die Anzahl der Inversionen im Vektor a. Bestimmen Sie den Erwartungswert von X; betrachten Sie dazu für $1 \le i < j \le n$ die binären Zufallsvariablen $X_{ij}$ mit $X_{ij}=1$ genau dann, wenn (i, j) eine Inversion ist. (Das Ergebnis ist die durchschnittliche Zahl der nötigen \glqq Swaps\grqq  beim Sortieralgorithmus Bubble Sort.)
