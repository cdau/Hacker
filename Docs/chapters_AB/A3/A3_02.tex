\subsection*{Aufgabe 2}
(Klausur SS 2012) Es seien A, B, C $\subset$ $\Omega$ Ereignisse in einem Wahrscheinlichkeitsraum. Geben Sie die Mengenschreibweise für folgende Ereignisse an:
\begin{enumerate}[leftmargin=1cm, label=\alph*)]
\item Es tritt A, aber weder B noch C ein.
\item Es treten genau zwei der drei Ereignisse ein.
\item Es tritt höchstens eines der drei Ereignisse ein.
\end{enumerate}
