\section{Aufgabe E33}
Sei X eine Zufallsvariable mit Dichte \newline\newline
$f(x)=\begin{cases}
  \dfrac{1}{3} \cdot x^2$ \hspace{10pt}für $ x \in [-1, 1]\\
  0$ \hspace{1.1cm}für x < 0 $\\
  \end{cases}$ 
Bestimmen Sie
\begin{enumerate}[leftmargin=1cm, label=\alph*)]
\item die Verteilungsfunktion von X,
\vspace{6cm}
\item den Erwartungswert und die Varianz von X,
\vspace{6cm}
\item $E[\frac{1}{X}]$
\end{enumerate}
