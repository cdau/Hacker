\newpage
Die Laktationsleistung bei einem weiblichen Rind aus einer speziellen Population (Milchleistung zwischen zwei Geburten, gemessen in kg) lasse sich ausreichend genau durch eine N(4000, $10^6$) - verteilte Zufallsvariable beschreiben.

\begin{enumerate}[leftmargin=1cm, label=\alph*)]

\item Berechnen Sie die Wahrscheinlichkeit, dass eine zufällig aus der Population herausgegriffene Kuh eine Laktationsleistung von
\begin{enumerate}[leftmargin=1cm, label=\roman*)]

\item weniger als 3000 kg hat
\vspace{4cm}
\item mehr als 6000 kg hat.
\vspace{4cm}
\end{enumerate}
\item Welche Laktationsleistung muss eine Kuh mindestens erbringen, um nicht zu den schlechtesten 33\% dieser Population (in Bezug auf die Laktationsleistung) zu gehören?
\newpage
\item Eine Herde aus Rindern dieser Population bestehe aus 100 Kühen. Welche Mittelwert und welche Varianz besitzt die Milchleistung dieser Herde, falls vorausgesetzt wird, dass die Laktationsleistungen der Kühe stochastisch unabhängig ist?
\end{enumerate}
