\section{Aufgabe E17}
Gegeben sei eine Urne mit 5 roten und 2 schwarzen Kugeln.
\begin{enumerate} [leftmargin=1cm, label=\alph*)]
	\item Wie groß ist die Wahrscheinlichkeit beim gleichzeitigen Ziehen von zwei Kugeln keine schwarze Kugel gezogen zu haben.
	\vspace{2cm}
	\item Wie groß ist die Wahrscheinlichkeit beim gleichzeitigen Ziehen von vier Kugeln höchstens eine schwarze gezogen zu haben.
\end{enumerate}
