\section{Aufgabe E13}
Es seien $A,B,C,D$ Ereignisse in einem Grundraum $\Omega$. Wie lauten die folgenden Ereignisse in Mengenschreibweise?
\begin{enumerate} [leftmargin=1cm, label=\alph*)]
	\item Es tritt \textbf{nur} das Ereignis B ein.
	\item Es treten genau zwei der vier Ereignisse ein.
	\item Es tritt höchstens eines der vier Ereignisse ein.
\end{enumerate}
