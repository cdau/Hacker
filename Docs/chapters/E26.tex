\section{Aufgabe E26}
Eine Fluggesellschaft nimmt Sitzplatzbuchungen für ein kleines Flugzeug mit 20 Plätzen entgegen. Da sie
aus Erfahrung weiß, dass jeder Passagier (unabhängig von den anderen Passagieren) mit Wahrscheinlichkeit $\frac{1}{3}$ nicht erscheint, nimmt sie 25 Anmeldungen entgegen.
\begin{enumerate} [leftmargin=1cm, label=\alph*)]
	\item Mit welcher Wahrscheinlichkeit kommen mehr Passagiere als das Flugzeug Sitzplätze hat?
	\vspace{10cm}
	\item Wenn Sie jedem Fluggast den Sie nicht transportieren können einen Betrag von 500 \euro \newline ausbezahlen, wieviel müssen Sie dann im Mittel als Entschädigung bezahlen?
\end{enumerate}
