\section{Aufgabe E32}
Sei X die Lebensdauer einer Glühbirne. Nehmen Sie an, dass X eine mit Parameter \nicefrac{1}{2} exponentialverteilte Zufallsvariable sei, d.h. die Dichte sieht wie folgt aus:\newline
\begin{center}
$f(x)=\begin{cases}
  \dfrac{1}{2} \cdot exp(-\dfrac{1}{2}x)$ \hspace{10pt}für $ x >= 0 \\
  0$ \hspace{2.4cm}für x < 0 $\\
  \end{cases}$
\end{center}
(wobei eine Einheit einem Jahr entspreche).
\begin{enumerate} [leftmargin=1cm, label=\alph*)]
 \item Mit welcher Wahrscheinlichkeit lebt die Glühbirne länger als 10 Jahre?
 \item Berechnen Sie den Median von X (also den Zeitpunkt andem die Wahrscheinlichkeit, dass die Glühbirne
 bereits kaputt gegangen ist, genau 50\% beträgt).
\end{enumerate}
