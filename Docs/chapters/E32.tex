\section{Aufgabe E32}
Sei X die Lebensdauer einer Glühbirne. Nehmen Sie an, dass X eine mit Parameter \nicefrac{1}{2} exponentialverteilte Zufallsvariable sei, d.h. die Dichte sieht wie folgt aus:\newline
\begin{center}
$f(x)=\begin{cases}
  \dfrac{1}{2} \cdot exp(-\dfrac{1}{2}x)$ \hspace{10pt}für $ x >= 0 \\
  0$ \hspace{2.4cm}für x < 0 $\\
  \end{cases}$
